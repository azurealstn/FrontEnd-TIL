# Head 정리

```html
<!DOCTYPE html>
```

- 위 태그를 사용하면 해당 파일의 문서 타입은 HTML5라는 것을 명시해주는 것이다.

```html
<html lang="en"></html>
```

- `lang`: 문서에서 사용하는 주언어 명시

```html
<meta charset="UTF-8" />
<meta name="viewport" content="width=device-width, initial-scale=1.0" />
```

- `charset`: 문자열 인코딩 설정
- `viewport`: 사용자의 디바이스 크기에 맞게 컨텐츠를 표시할 수 있도록 설정한다.
